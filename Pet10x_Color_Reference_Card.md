## ACCESSIBILITY QUICK CHECK

### Safe Combinations ✅
- White text on #FD9340 (large text 17pt+)
- #1F1F1F text on #FFFFFF
- #484444 text on #FFFFFF  
- #E67E28 text on #FFFFFF (all text sizes)
- #FFFFFF text on #34C759
- #FFFFFF text on #FF3B30
- #1F1F1F text on #FFF4EB

### Use with Caution ⚠️
- #FD9340 text on #FFFFFF (large text only)
- #2FBFB8 text on #FFFFFF (large text only)
- Small text on colored backgrounds

### Never Use ❌
- #787878 text on #F8F8F8 (too low contrast)
- #FD9340 text on #FFE8D6
- Light colors on light backgrounds

## USAGE RULES (AT A GLANCE)

### ✅ DO
- Use #FD9340 for primary CTAs
- Use #E67E28 for text links
- Maintain 4.5:1 contrast for small text
- Test in both Light & Dark modes
- Use semantic colors consistently
- Add icons alongside colors

### ❌ DON'T
- Use #FD9340 for body text
- Mix semantic color meanings
- Ignore dark mode
- Rely on color alone
- Overuse brand colors
- Skip contrast testing
